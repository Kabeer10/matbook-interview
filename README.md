# MatBook - Workflow Management System

## Tech Stack

- **Next.js** - React Framework
- **Turso (SQLite) with Drizzle ORM** - Database & ORM
- **NextAuth.js** - Authentication
- **tRPC** - End-to-End Type Safety
- **ShadCN + Tailwind CSS** - UI Components & Styling
